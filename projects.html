EEG Attention Classifier (2026)

Using Python, I built a Support Vector Machine (SVM) and a Random Forest Classifier to analyze EEG data and classify attention levels. I used the Overt and Covert Attention Dataset on Kaggle as a training set, which contains EEG recordings of subjects performing attention-related tasks. I preprocessed the EEG data by extracting relevant features such as power spectral density and time-domain. After training the models, I evaluated their performance using metrics like accuracy, precision, recall, and F1-score. The initial SVM achieved an accuracy of around 25%, while the Random Forest Classifier performed better with an accuracy of approximately 33%.

Since then, I have added several measures of optimizing the both models to improve their performance. I have implemented feature selection to the SVM and kernel selection to the RF, as well as tuned their hyperparameters and allowed for cross-validation. I also built a neural network that would use the results of the classification models to make more informed predictions. After which, the models will be tested by using data from a Cornell Arxiv's multi-label EEG dataset, "attention classification using EEG", and "EEG data for Mental Attention".

This project provided me with experience on data-processing and creating classification models in context of EEG data. I have learned so much from what EEG is, how to preprocess, extract features, and how to build and optimize machine learning models to classify attention levels based on EEG data. I've also spent a lot of time learning several python libraries, several types of machine-learning models, and synthezing all of this information to create And I hope to use what I learned here to contribute in the future to the field of brain-computer interfaces and neurotechnology, which I find fascinating.

Scam Detector Bot for Discord (2026)

Developed code for a Discord bot that detects potential scams in messages using several techniques. Using the discord-phishing-scam dataset, I trained a machine learning model to classify messages as scams or not based on features like message content and historical data. The bot uses natural language processing (NLP) techniques to analyze message content and identify common scam patterns, such as suspicious links or phrases. Additionally, the bot incorporates a spam detection system that flags repeated messages in multiple channels, which is a common tactic used by scammers to spread their messages. This project provided me with experience in building a practical application of machine learning for real-time scam detection.
